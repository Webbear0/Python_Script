# Python_Script 工作区总览

本项目目录包含了在学习 Python 期间编写和保存的多个实战练手项目，涵盖了数据库应用、网络爬虫、自动化运维以及 Web 框架等多个领域。

所有项目的代码均已在 **Python 3.12+** 环境下经过测试修复，能够正常运行。

---

##  数据库与业务系统模拟
*依赖: `pymysql` (MySQL), `pyodbc` (SQL Server)*

- **`bank_os.py` / `bank_os_sqlserver.py`**
  - **功能**：模拟银行核心业务系统（开户、销户、存款、取款、查询余额）。
  - **特色**：分别实现了基于 MySQL 和 SQL Server 双版本的数据库接入，所有 SQL 查询均使用参数化占位符，具备完整的防注入安全机制。
- **`campus_network.py`**
  - **功能**：模拟数字校园的注册和登录验证功能。
  - **特色**：通过数据库表存储用户名密码，实现了完整的鉴权闭环。

---

##  网络爬虫项目
*依赖: `requests`*

- **`meituan.py`**
  - **功能**：抓取美团网指定关键字的商家信息（店名、路名、评价、人均消费等）。
  - **特色**：支持按页码多页爬取，动态注入 cookies，将结果结构化存储至本地 CSV。
- **`qianchengwuyou.py`**
  - **功能**：爬取前程无忧网站的职位详细信息（薪资、经验要求、学历等）。
  - **特色**：接入 API 接口解析 JSON/正则数据，提取岗位关键标签存储为 CSV 文件。
- **`qidian_text.py`**
  - **功能**：根据小说ID爬取起点中文网免费小说章节。
  - **特色**：自动排版处理爬虫数据，按章节正文批量保存为本地 TXT。
- **`qingtingfm.py` / `ximalaya.py`**
  - **功能**：音频网站（蜻蜓FM、喜马拉雅）播客节目内容抓包下载。
  - **特色**：破解了蜻蜓 FM 的反爬（基于 `hmac-MD5` 算法逆向生成 `sign` 防抓包验签），批量导出 `.mp3` 文件。

##  系统运维与实用工具
*依赖: `paramiko`, `jieba`, `wordcloud`, `Pillow`*

- **`remote_command.py`**
  - **功能**：基于 SSH 协议的远程服务器运维工具。
  - **特色**：不仅支持远程发送终端指令并读取回显，还内建了 SFTP 客户端功能，支持双向文件传输与重命名。
- **`word_cloud.py`**
  - **功能**：文本自动化关键词提取和词云生成工具。
  - **特色**：结合 `jieba` 分词剔除停用词（支持人名提取），基于频率生成高品质的词云海报图片，并通过 `tkinter` 进行 UI 预览。

##  Web 端开发
*依赖: `flask`*

- **`flask_project/` (Flask)**
  - **功能**：基于 Flask 生成的简单学校门户网站 DEMO。
  - **特色**：实现了路由跳转、静态资源（多媒体、图片）加载、模板引擎渲染（HTML 布局结构调整完善）。
