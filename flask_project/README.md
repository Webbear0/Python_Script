# django运行项目  
`pip install django`  
`django statrproject {项目名称}`  
`cd {项目名称}`  
`python manage.py startapp {应用名称}`  
`python manage.py runserver`  

# flask运行项目  
`pip install flask`  
`mkdir {项目名称}`  
`cd {项目名称}`  
`mkdir static {存放静态文件}` //存放静态文件  
`mkdir templates` //存放html文件  
`touch app.py`
